Migrate away from Easy Testimonials is the official migrator from Easy Testimonials to Strong Testimonials

== Description ==

Migrate your testimonials from Easy Testomonials to Strong Testimonials, a user-friendly and customizable WordPress testimonials plugin that’s responsive and doesn't require a single line of code to work. We offer an importing tool that makes it easy to move your testimonials to Strong Testimonials without making an effort to rebuild your testimonials.

== Which parts of my Strong Testimonials testimonials will automatically import to Strong Testimonials? ==

We’ll import the following customization from Easy Testimonials Gallery:

* Testimonial title
* Testimonial content
* Testimonial image
* Testimonial email
* Testimonial name
* Testimonial star rating
